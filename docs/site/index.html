C support library for raylib-style apps

Small native UI with shared app plumbing.

Flint keeps a raylib-compatible public surface available through flint.h, then adds UI controls, layout helpers, text rendering, themes, embedded assets, locale loading, file dialogs, runtime asset downloads, and Lyra account/sync helpers.

Frame Lifecycle

For normal screen-space UI, call BeginUIFrame once per frame. It updates the layout viewport, UI viewport, DPI scale, and frame state with a safe default camera. 

BeginDrawing();
BeginUIFrame(GetScreenWidth(), GetScreenHeight(), GetUIScale());
SetUIColors(GetThemeText(), GetThemeBackground(),
              GetThemeSurface(), GetThemeCircle(),
              GetThemeButton(), GetThemeButtonHover(),
              GetThemeIcon());

If you need a transformed UI camera, call InitUI and SetUIFrame(camera). Flint normalizes invalid cameras, so a zeroed camera no longer breaks pointer input.

Modals

Use DrawUIActionModal for title, message, and action-button dialogs. It measures labels, fits button text, wraps long action rows, and registers modal input capture so background controls cannot receive clicks. 

UIModalAction actions[2] = {
    { "Keep running", UI_BUTTON_STYLE_PRIMARY, 0 },
    { "Quit", UI_BUTTON_STYLE_DANGER, 0 }
};

int result = DrawUIActionModal((UIModalSpec){
    .title = "Close app?",
    .message = "Choose what the app should do next.",
    .actions = actions,
    .action_count = 2,
    .close_icon = close_icon
});

Use DrawUIModalFrame when a dialog needs custom controls such as text fields, sliders, or scrollable content.

App Support

Flint is the source owner for shared application plumbing. Apps keep their build outputs local, while common dependency source and build recipes live under Flint so account management and network builds stay consistent.

Lyra Account

LyraAccount creates, imports, exports, signs, and verifies generic account keys without app-specific key names.

liboqs

vendor/liboqs is the shared source for post-quantum signing support. Downstream builds can enable it with HAS_LIBOQS.

curl + LibreSSL

vendor/curl is the shared source for app networking. TLS builds use curl's OpenSSL-compatible options through the OPENSSL_* make variables.

Fonts

Flint loads application-supplied TrueType or OpenType font assets. Load assets with LoadUIFontAsset, register the result with RegisterUIFont, then switch fonts with UseUIFont. 

Font font = LoadUIFontAsset("fonts/noto/NotoSans-Regular.ttf", UI_TEXT_BASE_SIZE);
RegisterUIFont("ui", font);
UseUIFont("ui");

C UI components for raylib apps

Stable primitives for small, native-feeling interfaces.

Flint provides frame lifecycle, modal capture, text input, buttons, dropdowns, checkboxes, theme colors, DPI scaling, and chopped bitmap font support behind a compact C API. 

flint_ui_begin_frame(
    GetScreenWidth(),
    GetScreenHeight(),
    flint_get_dpi_scale());

ui_set_colors(text, bg, surface,
              circle, button,
              hover, icon);

Start

Build Flint

make libflint.a

Include flint.h or specific headers from include/.

Link

-Lvendor/flint -lflint -lraylib -lGL -lm -lpthread -ldl -lrt -lX11

Use your platform's raylib/OpenGL flags as needed.

Frame Lifecycle

For normal screen-space UI, call flint_ui_begin_frame once per frame. It updates the layout viewport, UI viewport, DPI scale, and frame state with a safe default camera. 

BeginDrawing();
flint_ui_begin_frame(GetScreenWidth(), GetScreenHeight(), flint_get_dpi_scale());
ui_set_colors(flint_theme_get_text(), flint_theme_get_bg(),
              flint_theme_get_surface(), flint_theme_get_circle(),
              flint_theme_get_button(), flint_theme_get_button_hover(),
              flint_theme_get_icon());

If you need a transformed UI camera, call ui_init and ui_set_frame(camera). Flint normalizes invalid cameras, so a zeroed camera no longer breaks pointer input.

Modals

Prefer ui_draw_modal_frame for custom modal content. It draws the shared modal frame and registers modal input capture so background controls cannot receive clicks. 

FlintUIPanelFrame frame = ui_draw_modal_frame(
    flint_px(360), flint_px(260), "Edit",
    (Texture2D){0}, (Texture2D){0});

ui_draw_label_text_field(row, frame.content_x, frame.content_y, frame.content_w);

Fonts

Flint ships tools/otfchop for generating bitmap font atlases and metadata. Load generated assets with flint_text_load_chopped_font, then install the font with flint_text_set_font and flint_text_set_small_font. 

Font font = flint_text_load_chopped_font(
    "assets/fonts/locales.png",
    "assets/fonts/locales.dat",
    FLINT_TEXT_BASE_SIZE);
flint_text_set_font(font);

# Quick Script Reference

All Kryon utility scripts are located in the `scripts/` directory.

## 🌐 Web Development
- `./scripts/build-web-examples.sh` - Generate web examples from Kryon files
- `./scripts/build-web-examples.sh --serve` - Build and start development server

## 🏃 Running Examples  
- `./scripts/run_examples.sh` - Run all examples with default renderer
- `./scripts/run_examples.sh --renderer html` - Run with specific renderer

## 🧪 Testing
- `./scripts/run_tests.sh` - Quick test runner
- `./scripts/run_test_suite.sh` - Full test suite with coverage

## 📦 Building & Bundling
- `./scripts/bundle_krb.sh <app.kry>` - Bundle app for distribution

# Basic Kryon Application

A simple template demonstrating basic Kryon UI components.

## Features

- Window with title and size
- Text labels with different font sizes
- Button with click handler
- Container layout

## Running

```bash
kryon run main.kry
```

## Learning

This template shows:
- Basic window setup
- Text components
- Button components
- Event handling (onClick)
- Container-based layout
